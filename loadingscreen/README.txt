# Imperial RP Loading Screen (GitHub Pages)

## Quick Setup
1. Upload the contents of this folder to a GitHub repo (recommended name: `loadingscreen`)
2. Enable **Settings → Pages** → Deploy from branch → `main` → `/ (root)`
3. Your URL will be:
   `https://YOURNAME.github.io/loadingscreen/`

## GMod
Set in `server.cfg`:
`sv_loadingurl "https://YOURNAME.github.io/loadingscreen/"`

## Customise
Edit `config.json`:
- `discordInviteUrl`
- `rules`
- `staff`
- colours, tagline, tips
- optional `backgroundVideo` (upload mp4 into assets and set path)
